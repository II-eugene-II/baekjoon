각 줄 별로 입력받아서, 정수형으로 변환해준 후, 둘을 더해줍니다.

```
A = input()
B = input()

A = int(A)
B = int(B)

print(A+B)
```

더 짧게 하려면 다음도 가능합니다.

```
print(int(input())+int(input()))
```
